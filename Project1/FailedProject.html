What is JavaScript? History & Evolution

JavaScript is a dynamic scripting language used primarily for creating interactive and complex features in web pages and applications. It was developed in 1995 and has evolved into one of the most popular and influential programming languages worldwide.

What is JavaScript?

JavaScript enables the creation of interactive elements on web pages, such as animations, forms, and games. It runs mainly in browsers, but also on servers (via technologies like Node.js), and follows multiple programming paradigms, including object-oriented and functional styles.

  • Originally designed to be lightweight and interpreted for client-side scripting.
  • Used for manipulating HTML and CSS, handling user input, and communicating with web servers.

History

  • 1995: Brendan Eich at Netscape creates JavaScript in just 10 days, initially called "Mocha," then "LiveScript," and finally rebranded as "JavaScript" for marketing reasons.
  • 1996: Microsoft introduces JScript, its own JavaScript implementation for Internet Explorer, sparking the browser wars.
  • 1997: JavaScript is submitted for standardization, resulting in ECMAScript 1 published by ECMA International (the official specification for JavaScript).
  • 1999: ECMAScript 3 adds features like regular expressions and error handling.
  • 2005: AJAX emerges, revolutionizing dynamic web applications.
  • 2008: Google Chrome launches, introducing the fast V8 JavaScript engine.
  • 2009: Node.js brings JavaScript to server-side programming.
  • Over the years, JavaScript gained advanced features—modules, classes, improved syntax, and modern standards through ECMAScript updates up to ES2021.

Evolution

  • JavaScript’s rapid adoption led to fragmentation (multiple browser implementations), resolved by ECMAScript standardization.
  • Modern JavaScript supports complex application development, async programming, functional paradigms, and has an active ecosystem of frameworks (React, Angular, Vue).
  • Server-side JavaScript (Node.js, Deno) expanded its reach beyond browsers.
  • Today, JavaScript is a universal language for both client and server, powering most of the interactive web and a growing number of non-browser applications.
JavaScript's continued evolution, led by the TC39 committee and ECMAScript standards, makes it a cornerstone of modern software development.

ECMAScript & JavaScript Versions

Version Year Key Features
ES1 1997 First standardization.
ES2 1998 Editorial updates only.
ES3 1999 Regular expressions, try/catch, do-while, switch.
ES4 - Abandoned, never standardized.
ES5 2009 Strict mode, array methods, JSON, getters/setters.
ES6 (ES2015) 2015 let/const, arrow functions, modules, classes.
ES7 (ES2016) 2016 Exponentiation (**), Array.includes()
ES8 (ES2017) 2017 Object.values(), async/await, string padding.
ES9 (ES2018) 2018 Spread/rest for objects, Promise.finally()
ES10 (ES2019) 2019 Array.flat(), Object.fromEntries(), String.trimStart/End()
ES11 (ES2020) 2020 Nullish Coalescing (??), Optional Chaining (?.)
ES12 (ES2021) 2021 Logical assignment, String.replaceAll(), WeakRefs
ES13+ (2022-2025) 2022-2025 Top-level await, new syntax and features

Each ECMAScript version has modernized JavaScript with new syntax and capabilities for web, server, and application development, keeping the language at the center of software engineering.

JS file inclusion in HTML

To include a JavaScript file in an HTML document, use the <script> tag with the src attribute specifying the path to the JavaScript file. Here is the basic syntax: 

<script src="path/to/your/script.js"<</script>
  • Place this tag either inside the section or just before the closing tag.
  • Example inside <head>: 
    •       
        <head>
            <script src="js/script.js"></script>
        </head>
